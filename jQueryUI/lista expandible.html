Programación Web

HTML

Es un lenguaje utilizado para el desarrollo de páginas web.
Podemos utilizar cualquier editor de texto para editar o manipular el contenido de un documento HTML.

JavaScript

Es un lenguaje que nos permite crear pequeños programas que se insertan en páginas web.
JavaScript nos permite crear efectos dinámicos para nuestras páginas web.

jQuery

Son librerías o ficheros de JavaScript, que nos facilitan ejecutar tareas en páginas web.
jQuery nos permite implementar process ya listos para usar, facilitándonos la solución de tareas básicas.
